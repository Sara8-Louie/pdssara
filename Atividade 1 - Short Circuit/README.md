## O que é Short Circuit?

Short circuit é um comportamento dos operadores lógicos (&& e ||) onde a execução é interrompida assim que o resultado final já pode ser determinado.

## Aplicações desenvolvidas:

1. Verificação de login
2. Definição de valor padrão
3. Evitar erro ao acessar objeto
4. Execução condicional de função
5. Substituição de if simples

## Tecnologias utilizadas:
- JavaScript
- HTML
